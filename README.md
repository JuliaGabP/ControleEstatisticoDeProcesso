# Controle Estatístico de Processo (CEP) 

Este projeto é uma ferramenta desenvolvida para auxiliar no monitoramento e na análise de processos produtivos. O objetivo é identificar variações, garantir a estabilidade do processo e auxiliar na tomada de decisões baseada em dados estatísticos.

## Sobre o Projeto

O Controle Estatístico de Processo (CEP) é uma metodologia fundamental na gestão da qualidade. Este software permite visualizar se um processo está operando dentro dos limites de controle estabelecidos, ajudando a distinguir entre causas comuns e causas especiais de variação.

### Funcionalidades principais:

* **Cálculo de Médias e Desvios:** Processamento automático de dados inseridos.
* **Limites de Controle:** Definição de Limite Superior de Controle (LSC) e Limite Inferior de Controle (LIC).
* **Visualização Gráfica:** Geração de gráficos de controle para análise visual imediata.
* **Interface Intuitiva:** Facilitando o uso por operadores e gestores de qualidade.

## Como executar

Para rodar este projeto em sua máquina local, siga os passos abaixo:

1. **Clonar o repositório:**
   ```bash
     git clone [https://github.com/JuliaGabP/ControleEstatisticoDeProcesso.git](https://github.com/JuliaGabP/ControleEstatisticoDeProcesso.git)
   ```

2. **Acessar o diretório:**
  ```bash
    cd ControleEstatisticoDeProcesso
  ```

3. **Instalação de dependências:**
  ```bash
    pip install -r requirements.txt
  ```

4. **Execução:**

  ```bash
    python app.py
  ```
